# Erion Ember Color Palette & Typography

This document defines the visual design tokens for the Erion Ember brand, derived from `theme.css`.

## Color Palette

### Primary (Amber)
- **Hex**: `#f59e0b`
- **RGB**: `245, 158, 11`
- **Usage**: Primary brand color, logo accents, key interactive elements.

### Radiant Accent
- **Hex**: `#fffbeb` (Light Mode) / `#92400e` (Dark Mode)
- **Usage**: Subtle backgrounds, glowing effects, secondary logo elements.

### Foundation
- **Dark (Charcoal)**: `#171717` (Background), `#262626` (Cards)
- **Light (White)**: `#ffffff`
- **Text**: `#262626` (Light Mode), `#e5e5e5` (Dark Mode)

## Typography

### Headings & UI
- **Primary**: `Inter, sans-serif`
- **Style**: Modern, clean, high readability.

### Detail & Editorial
- **Secondary**: `Source Serif 4, serif`
- **Style**: Refined, technical yet approachable.

### Code & Technical
- **Monospace**: `JetBrains Mono, monospace`
- **Style**: Optimized for developers and technical documentation.
